    • 사용자 점수 보관 기능

    각 게임 화면에 점수 표기란을 만들고, 사용자가 게임에서 점수를 획득하면 해당 점수를 화면에 기록합니다. 학생들은 로컬에서 점수를 유지하는 방법을 고민해야 하지만, 실제 점수 저장은 JavaScript 없이 시각적으로만 구현됩니다.

    게임 페이지

    게임 페이지는 나중에 JavaScript로 만들어지는 게임이라고 가정합니다 (지뢰 찾기, 벽돌 깨기, 갤러그 게임 등). 각 게임에는 "다시 하기" 또는 "메인 화면으로 돌아가기" 버튼을 배치합니다.
